Git Tips
========

Add the list of files
```git
$ git add <list of files>
```

Add all files
```git
$ git add -a
```

Add all txt files in current directory
```git
$ git add *.txt
```

Add all txt files in docs directory
```git
$ git add /docs*.txt
```

Add all txt files in the whole project
```git
$ git add "*.txt"
```
